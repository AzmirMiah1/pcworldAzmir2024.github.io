Internal System Unit Components

The system unit contains essential components for the computer’s functionality. Key components include:


Elements of a typical or general computer include the Central Processing Unit, the Main Memory and various Input/Output devices. The CPU is responsible to perform all operations of a computer like calculating, receiving instructions and other operations. It interacts with the main memory (RAM) for reading and writing operations to be performed at fast rates. They are used to keep data temporarily so as to enhance the processing of information from the CPU. The data from input devices such as the keyboards and mouse are input to the CPU while the result from output devices such as the monitor and printers is output from the CPU. Several elements make up a system bus: address, data and control which allow for the interconnection of these components. The system bus makes it possible for data, command, as well as memory addresses, to move freely within the computer with the right interaction between the parts.


  • CPU (Central Processing Unit): Often considered the "brain" of the computer, it processes instructions from software and manages tasks through various processing units.
  • Motherboard: The main circuit board that connects all hardware components, including the CPU, RAM, and storage devices, enabling communication between them.
  • RAM (Random Access Memory): Temporary storage that holds data for active tasks, allowing quick access and efficient multitasking.
  • Power Supply Unit (PSU): Converts electricity from an outlet into usable power for the computer and distributes it to components.
  • Storage Drives (HDD/SSD): Permanent storage for data and applications. SSDs are faster than HDDs due to their lack of moving parts.

Roles of the ALU, Control Unit, Clock, and Registers

Arithmetic Logic Unit (ALU): Performs arithmetic and logical operations required by programs.

Control Unit: Directs operations within the CPU, managing data flow between components.

Clock: Synchronizes the timing of all processes in the CPU, measured in cycles per second.

Registers: Small storage locations within the CPU that hold data for quick access during processing.

Cache and Virtual Memory


Cache, Virtual memory Cache memory is a special high speed storage location within the CPU , which stores the most frequently used data and instructions in the computer. This decreases the usage of the main memory to obtain data for the CPU, thus improving its computational efficiency. Virtual memory comes into play when the physical memory in your machine is not sufficient. It opens an area on the hard disk and provides additional space for the main memory thus making the system capable of undertaking bigger jobs or several programs. By using virtual memory there is also capability to undertake more multitasking, however it is significantly slower than the RAM due to time it takes to access the hard disk. Both cache and virtual memory means that a computer will be in a position to run even under high demand.


Cache Memory: A small amount of fast memory within the CPU to temporarily store data for quick access. It speeds up the processing by avoiding repeated access to slower main memory.

Virtual Memory: A portion of storage that functions as extra RAM. When the RAM is full, the computer stores data in virtual memory, slowing down the system but enabling multitasking beyond the physical limits of RAM.

Backing Storage Types and Performance Factors


Long-term data storage is supported by backing storage. They include hard disk drive (HDDs), solid state drives (SSDs), pen drives, optical media (CD’s, DVD’s), flash memory cards. The performance criteria include the rate at which data can be transferred either to be read or to be written and the total amount of data that can be transferred at a time. The advantages of SSDs are higher transfer rates and enhanced performance as compared with HDDs, but it is followed by higher cost. Optical media are: cheap, access is possible only by their removal from the computer but their access is slow. The flash memory cards and pen drives are mobile and available in different capacities for temporary document transfer and storage.
